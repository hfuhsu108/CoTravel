## 共用規範

- 平台：PWA，手機直式為主，兼顧桌面寬版。
- 風格：簡潔直覺、資訊密度高、現代清爽、淺色為主；情侶旅遊感但不幼稚。
- 「該趟旅程內」的畫面（②③④⑤）共用底部導覽列：**地圖｜文件｜行李**，以及返回旅程列表的入口。
- 全 App 共用一套主色（以 prototype 為準）。

## 導覽流程

```
登入/配對(0) → 旅程列表(1) → 點某趟 → 主畫面/地圖(2)
                                          ├─ 詳情(3)
                                          ├─ 文件匣(4)
                                          └─ 行李清單(5)
```

---

## 畫面 0 · 登入 / 配對

- 置中 Logo + App 名 + 標語（例「兩人一起，把行程排好」）
- 登入（Email 或 Google）
- 首次：建立配對 / 以邀請碼加入伴侶

## 畫面 1 · 旅程列表頁（App 入口）

- 頂部：左 Logo，右使用者頭像（設定/登出）
- 清單依狀態分三組：**進行中** → **即將出發**（依出發日近到遠）→ **過往旅程**（可摺疊）
- 旅程卡：封面縮圖、旅程名、日期區間、狀態/倒數標籤、兩人頭像
- 右下浮動「＋」→ 建立新旅程表單（名稱、目的地、起訖日、邀請同伴）
- 狀態：(a) 有資料正常態；(b) 空狀態（插圖 + 引導 + 建立第一趟按鈕）

## 畫面 2 · 主畫面（地圖 + 按天側欄）

- 上方標題列：左=旅程名+日期；中=日期分頁 Day1/2/3（目前日高亮）；右=兩人在線頭像 + 改動通知鈴鐺（紅點）
- 地圖三標記：
  - **定點**：帶序號實心 pin
  - **區域**：半透明圓 + 中央標籤，圓內小候選點
  - **書籤**：顏色區分「想去 / 已排入」
  - 相鄰定點間虛線交通路線（可開關）
  - 點標記浮出小卡：名稱、照片、評分、「加入某天 / 看文件」
- 側邊按天面板（可收合）：定點卡 / 區域卡 / 交通列 混排，可拖拉，底部「＋ 加項目」
  - 定點卡：序號、名稱、可選時間、縮圖、連文件小圖示
  - 區域卡：圓圈圖示、區域名、時段、候選數（可展開）
  - 交通列：交通圖示 + 預估時間，可點開看路線或改自定義
- 底部：搜尋列「搜尋景點…」+「地圖/清單」切換；搜尋結果可「加為定點 / 加入書籤」
- 狀態：(a) 側欄展開/收合；(b) 選中項目高亮、其他變淡；(c) 改動通知紅點+橫幅

## 畫面 3 · 地點 / 區域 / 交通 詳情（三型共用版型）

- **定點**：大圖、名稱、評分、地址、營業時間、可選造訪時間、備註、「連結文件」、「導航（開 Google Maps）」、「移到其他天 / 移除」
- **區域**：區域名、時段、範圍縮圖；候選店家清單（不排序，每個可勾「今天就去」）；新增候選、備註
- **交通**：起→終、方式切換（步行/大眾運輸/開車/自定義）、預估時間距離、路線縮圖；自定義可填方式+時間+備註，可連車票文件

## 畫面 4 · 文件匣

- 頂部四分類頁籤：**機票｜住宿｜文件｜其他**
- 文件項目：縮圖/圖示、檔名、類型、上傳者頭像、「離線可用」標記、若已連行程則顯示連結標籤
- 右下「＋ 上傳」：選分類、可連到某行程項目
- 狀態：(a) 正常態；(b) 某分類空狀態；(c) 離線模式（只顯示已快取，未快取變灰）

## 畫面 5 · 行李清單

- 頂部兩分頁：「我的行李」「對方的行李」（互看）
- 清單：勾選框 + 品名 + 可選分類標題；已勾=已打包
- 整體完成度（如 12/20）
- 右下「＋ 新增項目」
- 狀態：(a) 自己的可編輯勾選；(b) 對方的唯讀但可見進度
