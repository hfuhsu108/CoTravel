# 更新紀錄

本檔記錄「同行（CoTravel）」各版本的重點變更。版本號遵循語意化版本（SemVer）。

## v1.1.0 — 2026-06-27

### 新增
- **提醒功能（Web Push）**：可為景點、航班設定提醒，到期時即使 App 關閉也能收到手機系統通知。
  - 架構：Supabase `reminders` / `push_subscriptions` 表 + Edge Function `check-reminders` + pg_cron 每分鐘掃描 + Web Push 發送。
  - 內建模板：餐廳訂位、線上劃位、機場提早抵達、登機、退房、自訂。
  - 提醒時間可用固定選項、自訂提前量（任意數字＋分/時/天），或指定絕對時刻。
  - 設定頁新增「啟用推播通知」；夥伴設定的提醒透過 Realtime 即時同步。
- **書籤詳情編輯**：點書籤列表的景點可開啟詳情頁，編輯別名、清單、備註、文件連結，並可加到行程或移除書籤（不含時間設定）。

### 修正
- 線上版設定頁誤顯示「不支援推播通知」：部署流程補注入 `VITE_VAPID_PUBLIC_KEY`，推播訂閱得以正常寫入。

## v1.0.0 — 2026-06

首個正式版。共編旅遊行程 PWA，含地圖行程規劃、書籤收藏、文件匣（可離線檢視）、行李清單、雙人即時同步與改動通知、PWA 安裝與手動更新。
